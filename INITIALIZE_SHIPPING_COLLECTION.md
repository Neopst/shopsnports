## Collection Structure

### Document Schema

```javascript
{
  // ===== Identifiers =====
  id: string,                      // Firestore document ID
  requesterId: string,             // Firebase UID (required)
  affiliateId: string | null,      // Partner ID (optional)

  // ===== Client Information =====
  clientName: string,              // Customer/company name (required)
  clientEmail: string,             // Email address (required)
  clientPhone: string,             // Phone number (required)

  // ===== Shipment Details =====
  type: 'air' | 'sea' | 'land',   // Shipping method (required)
  status: 'pending' | 'approved' | 'inTransit' | 'delivered' | 'rejected' | 'cancelled',
  priority: 'economy' | 'standard' | 'express' | 'urgent',

  // ===== Route =====
  origin: string,                  // Departure city (required)
  destination: string,             // Arrival city (required)
  description: string,             // Item description (required)

  // ===== Dimensions & Weight =====
  weight: number,                  // kg (required)
  length: number,                  // cm (required)
  width: number,                   // cm (required)
  height: number,                  // cm (required)

  // ===== Costs (USD) =====
  estimatedCost: number,           // Estimated price (required)
  actualCost: number,              // Actual price (0 until delivered)
  affiliateCommission: number,     // Commission (0 if no affiliate)

  // ===== Delivery Timeline =====
  estimatedDelivery: Timestamp,    // Expected arrival (optional)
  actualDelivery: Timestamp,       // Actual arrival (optional)

  // ===== Insurance & Customs =====
  requiresInsurance: boolean,      // Insurance needed (required)
  requiresCustomsClearance: boolean, // Customs clearance (required)

  // ===== Tracking =====
  trackingNumber: string | null,   // Format: SHP-YYYYMMDD-XXXXX (auto-generated)

  // ===== Metadata (Server-Generated) =====
  createdAt: Timestamp,            // Document creation time
  updatedAt: Timestamp             // Last update time
}
```

## Manual Creation (Firebase Console)

If you prefer to create the collection manually:

### 1. Create Collection

1. Run this query in Cloud Firestore to create the collection:
   
   **Via Firebase CLI:**
   ```bash
   # Login
   firebase login
   
   # Set project
   firebase use shopsnports
   
   # Create empty collection marker (add then delete)
   firebase firestore:delete shippingRequests --all 2>/dev/null || true
   ```

2. Or create via Console:
   - Go to [Firebase Console](https://console.firebase.google.com/project/shopsnports/firestore/data)
   - Click **Start Collection**
   - Collection ID: `shippingRequests`
   - Skip auto-ID and manual document (leave empty)
   - Click **Save**

### 2. Create Indexes (Pre-existing)

✅ Already deployed. No action needed.

Verify at: [Firestore Indexes](https://console.firebase.google.com/project/shopsnports/firestore/indexes)

## Troubleshooting

### "serviceAccountKey.json not found"

**Fix:** 
1. Get service account key from Firebase Console
2. Place in `functions/serviceAccountKey.json`

```powershell
# Verify file exists
Test-Path functions\serviceAccountKey.json
```

### "Collection does not exist" after initialization

**Expected behavior:** Collection is initially empty. 
- Submit a shipping request from mobile app to populate it

### "Cannot read property 'toDate' of null"

**Cause:** Firestore timestamps not properly formatted
**Fix:** Use `admin.firestore.Timestamp.fromDate()` for date fields

### "Missing required fields" error when submitting request

**Fix:** Check [Field Definitions](#field-definitions) table above
- Ensure all ✅ fields are included
