2. End-to-End Flow Analysis

NOTIFICATION FLOWS

Flow 1: Push Notification to User
Admin/System → sendPushNotification →
Get FCM tokens → Send multicast →
Track success/failure → Cleanup failed tokens →
Log statistics

Flow 2: Email Notification
Shipping status change → shipping_email_notification_service →
Build HTML template → Call Cloud Function →
Send via SMTP → Log delivery

Flow 3: Notification Tap Handling
User taps notification → FirebaseMessaging.onMessageOpenedApp →
Extract data → Navigate to appropriate screen

INVOICE FLOWS

Flow 1: Auto-Generate Invoice on Delivery
Shipment delivered → generateInvoice →
Determine invoice type (affiliate/service) →
Get commission data → Create invoice document →
Update shipping request → Notify admin

Flow 2: Manual Invoice Generation
Admin clicks generate → generateInvoiceHttp →
Get shipping request → Create invoice →
Return invoice details
